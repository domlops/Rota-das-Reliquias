Loop de jogo

  • Mover o personagem pelo mapa.
  • Coletar reliquias em pontos marcados.
  • Consultar a melhor rota entre objetivos restantes.
  • Organizar o inventario por peso, valor ou raridade.

Atalhos planejados

  • WASD / Setas: mover
  • E: coletar item
  • I: abrir inventario
  • R: mostrar ou recalcular rota

Teste desta etapa

  • Use `WASD` ou as setas para mover o jogador.
  • O marcador verde deve permanecer dentro do mapa.
  • O painel deve atualizar coordenadas e direcao.
  • Use `Novo jogo` para redistribuir base, itens e inicio.
  • Fique perto de um ponto e pressione `E` para coletar.
  • Pressione `R` para mostrar a rota sugerida e recalcula-la quando necessario.
  • Pressione `I` para abrir e fechar o inventario.
  • Use `Ordenar por` no inventario para alternar o criterio de exibicao dos itens.
